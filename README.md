STUDENT PERFORMANCE DASHBOARD

An AI-powered web application built using Flask and Machine Learning that helps students analyze their academic performance, predict future outcomes, and improve their study patterns through data-driven insights.

---

FEATURES

• Student Report Analyzer

* Input marks and calculate SGPA/CGPA
* Visualize performance using charts

• Prediction System

* Predict future academic performance using ML models
* Uses scikit-learn for training and prediction

• Prediction Comparer

* Compare predicted vs actual results
* Get improvement suggestions

• Class Timetable

* Weekly schedule management
* Upload and download timetable as image

• Student Lifestyle Fixer

* Helps balance daily routine
* Detects overlaps and suggests improvements

• Mental Support 24/7

* Basic support section for student well-being

HOW TO RUN LOCALLY

1. Clone the repository

2. Install dependencies:
   pip install -r requirements.txt

3. Run the Flask app:
   python app.py

4. Open browser and go to:
   http://127.0.0.1:5000

---

DEPLOYMENT

This project is deployed using Render with Gunicorn.
